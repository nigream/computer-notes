# 常用命令

## 关于shell

### 不同实现

- 在不同的Linux系统中，有不同的shell 。
- 如 Linux 默认的 bash、 debian 的 dash等。
- bash (Bourne-Again SHell) 源自 UNIX 的 Bourne shell 。

### sh

- 打开 shell 其实就是执行了 `/bin` 目录中的 sh 命令

- 我们查看 CentOS 下的 `sh` 命令的属性，可以看到它链接到了同目录下的 `bash` 命令。
