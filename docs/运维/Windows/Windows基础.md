## 常用 DOS 命令

|       命令       |    操作符号     |
| :--------------: | :-------------: |
|     盘符切换     |    ``盘符:``    |
|  查看当前文件夹  |     ``dir``     |
|    进入文件夹    | ``cd 文件夹名`` |
|    退出文件夹    |    ``cd ..``    |
| 退出到磁盘根目录 |    ``cd \``     |
|       清屏       |     ``cls``     |
