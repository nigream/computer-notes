## 概述

- Appuploader 可以辅助在 Windows、Linux 或 MacOS 直接申请 IOS证书 p12 ，及上传 ipa 到 App Store 。
- 为没有苹果 Mac 电脑，而在 Windows 开发 IOS 的开发者提供方便。
