# 安装Python

1. 安装Python，自定义安装路径，一路next（最后我点击了 `Disable Path length limit` ）。

   ![image-20220117193600349](准备开发环境/image-20220117193600349.png)

   

2. 配置环境变量。

   path 变量后面加上 Python 的安装路径。

   

3. 验证是否安装成功。

   ```shell
   # 命令行输入（注意 V 是大写）
   python -V
   # 输出
   Python 3.10.1
   ```
